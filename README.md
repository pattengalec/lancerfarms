# Lancer Farms & Gardens

Live site: [https://www.lancerfarms.com](https://www.lancerfarms.com)

The working farm and garden management app for California Baptist University. Guests can browse beds, leave comments, and view photos; staff can log in to manage beds, track tasks, and record field activity.

## Stack

- GitHub Pages (static hosting)
- Vanilla HTML/CSS/JS (single-page app, all JS inline in `index.html`)
- Google Apps Script backend
- Google Sheets data store
- Cloudinary photo storage

## Branches

`main` is always the live site. New features are built on their own branch and merged in once tested.
